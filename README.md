### Back-end
* Express js
* Axios

### Installig
backend/
```
pnpm run install
```
client/
```
pnpm run install
```

### Env
backed/.env
```
VITE_GITHUB_CLIENT_ID=
VITE_GOOGLE_CLIENT_ID=
```
client/.env
```
GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
```

### Running app
backend/
```
pnpm run dev
```
client/
```
pnpm run dev
```
